Тобі дано {% if is_js %}масив{% else %}список{% endif %} цілих чисел data. Заміни кожен елемент у ньому найбільшим елементом серед елементів праворуч (не враховуючи поточний елемент). Останній елемент повинен бути замінений на -1. Поверни модифіковану послідовність як ітеративний об'єкт.

Поглянь на наступний приклад: 

data = [17, 18, 5, 4, 6, 1]
data1 = list(replace_biggest(data))
print(data1)

Вивід:

[18, 6, 6, 6, 1, -1]

Пояснення:

  • data1[0] --> найбільший елемент праворуч від data[0] це елемент з індексом 1 - 18;
  • data1[1] --> -//- з індексом 4 - 6;
  • data1[2] --> -//- з індексом 4 - 6;
  • data1[3] --> -//- з індексом 4 - 6;
  • data1[4] --> -//- з індексом 5 - 1;
  • data1[5] --> немає елемента праворуч від елемента з індексом 5, тому ти пишеш -1.
